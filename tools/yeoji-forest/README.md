# Yeoji Forest (여지의 숲)

> **"좋은 것과 나쁜 것을 가리지 않고 모두 품어, 그 위에 만물이 자라게 하는 대지(大地)"**
> 화엄경 십지품의 '여지(如地)' 정신을 코드 에디터로 옮겨왔습니다.

`Yeoji Forest`는 개발자가 마주하는 긴 시간 동안, 숲의 편안함을 느끼며 코드를 있는 그대로 담아낼 수 있도록 설계된 **다크 테마**입니다.

## 🌿 왜 Yeoji Forest 인가요?

개발자의 눈은 소중합니다. `Yeoji Forest`는 단순히 어둡기만 한 테마가 아닙니다.

* **차분함의 미학:** Everforest의 깊은 숲을 닮은 배경색을 기반으로, 오랜 시간 코드를 봐도 눈이 피로하지 않도록 명암비를 최적화했습니다.
* **또렷한 시인성:** 배경은 차분하지만, 코드의 키워드, 함수, 변수들은 채도 높은 색감으로 처리하여 복잡한 로직 속에서도 길을 잃지 않도록 도와줍니다.
* **포용적인 환경:** JSON, Markdown, 그리고 우리 'Hun' 프로젝트의 어셈블리까지, 어떤 코드를 담아도 숲이 만물을 품듯 편안하게 보여줍니다.

## 🚀 설치 방법

1. VS Code 확장 프로그램 탭(Ctrl+Shift+X)에서 **Yeoji Forest**를 검색하세요.
2. 설치 후 `Ctrl+K Ctrl+T` (Mac: `Cmd+K Cmd+T`)를 누릅니다.
3. 목록에서 **Yeoji Forest**를 선택하면 바로 적용됩니다.

## 🎨 주요 특징

* **가독성 중심:** 코드의 구조가 한눈에 들어오도록 각 요소(Keyword, Function, String, Type)를 명확하게 색상 구분했습니다.
* **범용성:** Web 개발(JS/TS, CSS)부터 시스템 프로그래밍(ASM, C/C++)까지 세밀한 문법 하이라이팅을 지원합니다.
* **조화로운 색감:** 눈에 거슬리는 자극적인 원색을 배제하고, 자연에서 영감을 받은 배색을 사용했습니다.
