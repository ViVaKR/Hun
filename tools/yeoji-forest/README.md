# Yeoji Forest (여지)

여지(如地) — 화엄경 십지품에서, 좋은 것 나쁜 것 가리지 않고 모두 품어 그 위에 만물이 자라게 하는 대지를 뜻합니다.
숲의 색감을 바탕으로, 코드를 있는 그대로 편안하게 담아내는 다크 테마를 만들고 싶었습니다.

Everforest의 차분한 배경 위에, 조금 더 또렷한 글자색을 얹었습니다.

## 설치 후

`Ctrl+K Ctrl+T` (Mac: `Cmd+K Cmd+T`) → **Yeoji Forest** 선택

## 특징

- 어두운 숲을 닮은 배경, 눈에 편안한 명암
- 키워드·함수·문자열·타입이 뚜렷하게 구분되는 채도 높은 강조색
- JSON, Markdown 등 다양한 파일 형식에 세밀하게 대응하는 문법 강조
